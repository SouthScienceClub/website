Resources

Here is a list of helpful resources to get you started!


  • Official Science Olympiad Website: soinc.org
  • Click here to directly go to the listing of all 2027 Division C events.

  • Science Olympiad Wiki Page: scioly.org/wiki

  • South's Build Manual (2025-26): Google Docs

  • South's Inquiry Guide (2025-26): Google Docs
  • This is a detailed guide to the inquiry events from the 2025-2026 season.
